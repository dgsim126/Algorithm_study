## 📁 폴더 구조
```
Algorithm_study/
├─ shipleaf/              # 참여자 이름(닉네임)
│  ├─ 2501/               # 년월 (예: 2025년 1월 → 2501)
│  │  ├─ 250101/          # 년월일 (2025.01.01)
│  │  │  └─ BOJ_1234.py   # 문제 이름
│  │  ├─ 250102/
│  │  │  └─ PGS_42576.py
│
├─ honggildong/
│  ├─ 2501/
│  │  ├─ 250101/
│  │  │  └─ BOJ_5678.py
```
## 📌 파일 작성 규칙
1️⃣ 참여자 폴더 생성
- 본인 닉네임/이름으로 최상위 폴더 생성

2️⃣ 년월 폴더
- 형식: YYMM
- 예: 2501 (2025년 1월)

3️⃣ 년월일 폴더
- 형식: YYMMDD
- 예: 250101

4️⃣ 파일명 규칙
- 플랫폼_문제번호.확장자
- 예시: BOJ_1234.py, PGS_42576.java
